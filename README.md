# VC_EBV_sheep
Ejemplo de juguete de valoración genética y estimación de componentes de varianza usando programas de la Familia BLUPf90  

# Nota
Para correr los ejemplos en Windows descargue los programas renumf90 y blupf90+ de https://nce.ads.uga.edu/html/projects/programs/Windows/64bit/

# Para correr los programas en CMD de Windows escribe (Ejemplo para un modelo unicaracter)
./renumf90 param_BLUP_1t.txt
./blupf90+ renf90.par
